# Registration Form

Одностраничная форма регистрации с клиентской валидацией на чистом JavaScript.  
Соответствует требованиям тестового задания: все поля валидируются в реальном времени, кнопка отправки разблокируется только при полностью корректных данных.

## Функциональность

- Поля: Имя, Фамилия, Email, Пароль, Подтверждение пароля, Дата рождения.
- Валидация при потере фокуса (`blur`) и при вводе (`input`).
- Требования к полям:
  - Имя/Фамилия: только буквы и дефис, длина 2–30 символов.
  - Email: корректный формат.
  - Пароль: минимум 8 символов, цифра, заглавная/строчная буква, спецсимвол.
  - Подтверждение пароля: совпадение с паролем.
  - Возраст: не младше 18 лет (проверка по дате рождения).
- Визуальная индикация: классы `valid`/`invalid` для полей и формы, сообщения об ошибках.
- Кнопка отправки блокирована (`disabled`), пока форма невалидна.
- Предотвращение отправки формы (демонстрационная заглушка).

## Технологии

- HTML5 (семантическая разметка, тип `email`, `date`, `password`)
- CSS3 (адаптивный дизайн, индикация состояний)
- JavaScript (чистый JS, без фреймворков, модульная структура)

## Структура проекта

## Запуск

Просто откройте `index.html` в любом современном браузере (Firefox, Chrome, Edge).  
Все стили и скрипты подключены локально, интернет не требуется.
